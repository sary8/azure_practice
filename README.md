# AZ-305 Exam Lab

Microsoft Learnの2026年4月17日版AZ-305スキル領域に沿って構成した、オリジナル問題65問の静的Webアプリです。実試験問題やダンプは収録していません。

## 重点

- サービス名を知っているだけではなく、互換性、運用責任、障害単位、RPO/RTO、コスト、ネットワーク境界などを比較して解く設計問題
- 複数の選択肢が要件の大半を満たし、最後の制約で最適解が決まる構成
- 共通資料を参照するケーススタディ
- 単一選択、複数選択、対応付け、ドロップダウン、リスト作成、ホットエリア、Yes/No判定
- 前提知識、制約の優先順位、回答のポイント、導出手順、最も近い誤答、条件変更時の別解、トレードオフを含む解説
- iPhoneのタップ操作に対応

## GitHub Pagesで公開する

1. このフォルダーの `index.html` を公開リポジトリのルートへ配置します。
2. GitHubのリポジトリで **Settings → Pages** を開きます。
3. **Deploy from a branch** を選びます。
4. Branchを `main`、Folderを `/ (root)` にして保存します。
5. 表示された公開URLをSafariで開きます。

ビルドや外部ライブラリは不要です。回答履歴はブラウザのローカルストレージへ保存されます。
